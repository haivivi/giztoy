## 概述

端到端实时语音大模型支持实时语音交互，适用于智能助手、语音客服等场景。

## 功能特性

- ✅ 实时语音输入
- ✅ 实时语音输出
- ✅ 低延迟交互
- ✅ 多轮对话支持
- ✅ 支持移动端 SDK
