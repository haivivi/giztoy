## 概述

豆包语音是火山引擎提供的语音技术服务，包含多个产品线：

## 产品目录

### 语音合成（TTS）

| 产品 | 说明 | 文档 |
|------|------|------|
| 豆包语音合成2.0 | 大模型语音合成，支持流式/异步 | [tts2.0/](./tts2.0/) |
| 语音合成1.0 | 经典版语音合成 | [tts1.0/](./tts1.0/) |
| 声音复刻 | 定制音色复刻 | [voice-clone/](./voice-clone/) |

| 产品 | 说明 | 文档 |

| 大模型语音识别 | 流式/录音文件识别 | [asr2.0/](./asr2.0/) |

## 快速开始

根据需求选择对应产品：

- **短文本实时合成** → 豆包语音合成2.0 流式接口
- **长文本批量合成** → 豆包语音合成2.0 异步接口
- **定制音色** → 声音复刻
- **实时语音识别** → 大模型流式语音识别
- **录音文件转写** → 大模型录音文件识别
