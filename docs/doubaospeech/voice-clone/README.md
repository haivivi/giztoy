## 概述

声音复刻服务允许用户上传音频样本，创建专属定制音色。

## 复刻类型

| 类型 | 说明 | 音频要求 |
|------|------|----------|
| 极速复刻 | 快速创建，效果一般 | 10-30秒 |
| 标准复刻 | 平衡效果和速度 | 1-3分钟 |
| 专业复刻 | 高品质定制 | 5-10分钟 |
| ICL复刻 | 基于大模型的复刻 | 10-60秒 |

## 音频要求

- **格式**: WAV, MP3, M4A, FLAC
- **采样率**: ≥16kHz
- **位深**: ≥16bit
- **内容**: 清晰的人声，无背景音乐/噪音
- **语言**: 普通话、英语等

## 使用流程

1. 录制/准备音频样本
2. 调用复刻 API 上传音频
3. 等待音色创建完成
4. 在 TTS 接口中使用复刻音色
