## 概述

播客功能支持生成多人对话形式的音频内容，适用于播客节目制作、有声读物等场景。

## 功能特性

- ✅ 多角色对话生成
- ✅ 自动分配说话人
- ✅ 支持角色音色定制
- ✅ 自然的对话节奏
