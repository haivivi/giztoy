大模型语音识别（ASR 2.0）提供高精度的语音转文字服务。

## 功能特性

- ✅ 支持多语种识别
- ✅ 支持方言识别
- ✅ 支持标点恢复
- ✅ 支持说话人分离
- ✅ 支持热词定制
- ✅ 支持时间戳输出
