## 概述

豆包语音妙记是智能会议记录服务，支持实时语音转写、说话人识别、会议纪要生成等功能。

## 功能特性

- ✅ 实时语音转写
- ✅ 说话人识别（声纹）
- ✅ 自动生成会议纪要
- ✅ 关键词提取
- ✅ 待办事项识别
- ✅ 多语种支持

## 使用场景

- 会议记录自动化
- 访谈转写
- 培训记录
- 客服通话记录
