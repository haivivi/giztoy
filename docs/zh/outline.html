Giztoy 文档

Giztoy 核心包文档

本目录包含 Giztoy 所有核心包的设计文档、实现说明和已知问题。 库文档位于 docs/lib/ 目录下。

文档结构

每个模块在 docs/lib/ 下都有独立目录，包含以下文件：

lib/{mod}/
├── doc.md      # 主文档：设计目标、特性、用法示例
├── go.md       # Go 实现：接口设计、类型定义、使用方法
├── rust.md     # Rust 实现：trait 设计、类型定义、进度
├── issues.md   # 已知问题：bug、待改进、设计缺陷
└── {submod}/   # 子模块（如有）
    ├── doc.md
    ├── go.md
    ├── rust.md
    └── issues.md
