# 常见问题 Q&A

## 1. 使用测试环境还是正式环境适配？

**答**：直接使用正式环境适配。

## 2. 设备端如何使用九天大模型闲聊服务？

**答**：设备端先发送 AI 请求到厂商平台，厂商平台中控意图识别到 AI 请求为闲聊时，由厂商平台调用九天大模型接口处理，返回给设备端。

## 3. 设备端可以直接调用九天大模型接口吗？

**答**：支持，九天大模型接口支持端云和云云对接。如果想改为采用端云对接形式，请联系对接人并发邮件说明。

## 4. 请求头参数 X-AI-UID 和 X-AI-VID 如何获取？

**答**：

- **X-AI-VID**: 产品 ID (product_id)，在平台创建产品信息时生成
- **X-AI-UID**: 设备 ID (device_id)，在 AI 设备接入纳管，调用获取设备信息接口获得（请查阅终端智能体服务管理平台设备接入协议文档）

## 5. 模型上下文长度是多少，超过了怎么处理？

**答**：目前服务端模型最大支持 8K，模型上下文内容由客户端发送。为了避免异常情况，客户端请求大模型接口，如果上下文长度超过 8K 需要做一下截断。

## 6. 儿童手表等产品，需要接入非生成式 AI 怎么处理？

**答**：申请 token 的时候，在邮件里面注明需要非生成式 AI，非生成式 AI 的 model 参数使用 `jiutian-chat`。

详见 [Chat API 文档](chat.md#非生成式-ai)。
