豆包语音 SDK

火山引擎豆包语音 API 的 Go 和 Rust SDK。

官方 API 文档：api/README.md

设计目标

  1. 双 API 版本支持：V1（经典）和 V2/V3（大模型）API
  2. 多种认证方式：Bearer Token、API Key、V2 API Key
  3. 全面覆盖：TTS、ASR、声音克隆、实时对话、会议、播客等
  4. 流式优先：基于 WebSocket 的流式传输，适用于实时场景

API 版本

豆包语音有两代 API：

版本名称功能推荐
V1经典版基础 TTS/ASR旧项目
